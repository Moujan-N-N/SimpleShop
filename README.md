# SimpleShop

SimpleShop is a simple online shopping application built with ASP.NET Core MVC.

This project is being developed as a portfolio project to practice building a real-world e-commerce application using ASP.NET Core and related technologies.

## Technologies

- ASP.NET Core MVC
- .NET 10
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- Bootstrap
- C#

## Features

- User authentication and authorization
- Product management
- Category management
- Admin area
- Product image upload
- Shopping cart
- Product browsing
- Database integration with Entity Framework Core
